# To-Do List App

Простое To-Do приложение, созданное на Vanilla JavaScript.

## Возможности

* Добавление задач
* Удаление задач
* Отметка выполненных задач
* Фильтрация задач
* Сохранение в localStorage

## Технологии

* HTML5
* CSS3
* JavaScript (ES6)
