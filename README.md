# To-Do List App

A simple and clean To-Do List application built with Vanilla JavaScript.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Edit tasks inline
- Filter tasks by status
- Save tasks in localStorage
- Toggle light and dark theme

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage
